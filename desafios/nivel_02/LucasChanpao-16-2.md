### Exercício 16 [PHP BÁSICO];

* Tabuada em PHP;

~~~ php 
<?php

$multi = 8;

echo "<h2> TABUADA DO 8, Exercicio 16 </h2>"; // - 16
for ($i = 1; $i <= 10; $i++) {
    $tab = $multi * $i;
    echo $multi, "X", $i, "=", $tab . "<br>";
}
?>
~~~
